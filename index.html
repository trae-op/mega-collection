⚡ MegaCollection

Tree-shakeable search, filter & sort engine for 10 M+ item collections — import only what you need

🔍

Search

Trigram-accelerated substring search, plus O(1) exact key lookup via hash-map indexes.

O(1) exact · trigram contains 
import { Indexer, TextSearchEngine }
  from "@devisfuture/mega-collection/search";

const indexer = new Indexer();
indexer.buildIndex(users, "city");
indexer.getByValue("city", "Kyiv"); // O(1)

const search = new TextSearchEngine();
search.buildIndex(users, "name");
search.search("name", "john");
☑️

Filter

Multi-criteria AND filtering. Uses indexed lookups when available, Set-based fallback otherwise.

O(k) indexed · AND logic 
import { FilterEngine, Indexer }
  from "@devisfuture/mega-collection/filter";

const indexer = new Indexer();
indexer.buildIndex(users, "city");
indexer.buildIndex(users, "age");

const filter = new FilterEngine(indexer);
filter.filter(users, [
  { field: "city", values: ["Kyiv"] },
  { field: "age",  values: [25, 30] },
]);
↕️

Sort

Multi-field sorting powered by V8 TimSort with pre-compiled comparators and numeric index-sort optimisation.

O(n log n) · multi-field 
import { SortEngine }
  from "@devisfuture/mega-collection/sort";

const sorter = new SortEngine();
sorter.sort(users, [
  { field: "age",  direction: "asc" },
  { field: "name", direction: "desc" },
]);
